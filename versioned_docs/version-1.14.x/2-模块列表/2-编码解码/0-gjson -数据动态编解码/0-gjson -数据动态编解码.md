`gjson` 模块实现了强大的数据编码/解析功能，支持数据层级检索、动态创建修改 `Json` 对象，并支持常见数据格式的解析和转换等特点。

特点：

1. 支持数据层级检索；
2. 支持运行时数据修改；
3. 支持动态创建层级数据结构，并转换为支持的数据格式；
4. 支持 `JSON`、 `XML`、 `INI`、 `YAML/YML`、 `TOML`、 `Struct` 数据格式相互转换；

**使用方式**：

```  go
import "github.com/gogf/gf/encoding/gjson"

```
